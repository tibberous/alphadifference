AlphaDifference
GitHub Photoshop extension TriState.Digital

Pull out exactly what changed.

Upload two versions of an image - a before and an after - and AlphaDifference extracts only the pixels that changed, on transparency. The one-click Photoshop never shipped. Everything runs in your browser; your images never get uploaded anywhere.

Changed = keep the after's real colors. Difference = the delta colors.
Soft = alpha fades with how much changed. Hard = solid.
