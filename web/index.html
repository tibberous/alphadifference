The settings (everywhere)

  • Algorithm - Changed pixels: keeps the after image's real colors where they changed. This is the usual one (recover the torches as they actually look).
  • Algorithm - Difference: outputs the raw delta colors (|after - before|) - the classic "Difference blend" look.
  • Edge - Soft: alpha fades with how much each pixel changed - smooth, perfect for light/glow.
  • Edge - Hard: changed pixels are fully solid, the rest fully transparent.
  • Tolerance (0-100): ignore changes smaller than this. Bump it up to kill JPEG/anti-alias speckle.
  • Feather (px): soften/blur the cut edge.

Do it with ImageMagick (CLI)

Photoshop has no native button for this, but ImageMagick has had it for decades.

1. Install ImageMagick

Windows:  winget install ImageMagick.ImageMagick      (or get the installer at imagemagick.org)
macOS:    brew install imagemagick
Linux:    sudo apt install imagemagick

2. Run it - keep the changed pixels (true color) on transparency:

magick after.png before.png -fuzz 8% -compose ChangeMask -composite out.png
  • -fuzz 8% = the Tolerance (raise it to ignore small changes).
  • -compose ChangeMask = the Changed pixels algorithm. For the raw Difference instead: -compose difference.
  • Soft glow / feather: feather the alpha with -channel A -gaussian-blur 0x2 +channel; crop to the change with -trim +repage.

Prefer one command? Grab our Python wrapper (cli/alphadifference.py) from the repo - it exposes --mode, --fuzz, --feather, --trim and even auto-installs ImageMagick if it's missing.
